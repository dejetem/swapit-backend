## General

**Development mode**

- **Step 1:** `yarn install`
- **Step 2:** `yarn run dev` or `npm run dev`

**Production mode**

- **Step 1:** `yarn install`
- **Step 2:** `yarn build`
- **Step 3:** `yarn start`
